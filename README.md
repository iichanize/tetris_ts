# Tetris_ts

## 環境

|            | バージョン |
| ---------- | ---------- |
| Node.js    | 14.17.0    |
| TypeScript | 4.3.5      |

## 操作方法

| キー  | 動作                                |
| ----- | ----------------------------------- |
| ← / → | 左移動 / 右移動                     |
| ↓     | 高速落下                            |
| z / x | 左回転 / 右回転                     |
| a     | HOLD と落下中のテトリミノの入れ替え |
